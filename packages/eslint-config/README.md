# `@sonnymiel/eslint-config`

This package contains my customs `eslint-config`.

## Installation

```sh
npm install -D @sonnymiel/eslint-config
```

## Usage

```js
  import config from "@sonnymiel/eslint-config";

  export default [
    ...config.base,
    ...config.react,
    ...config.stylistic,
  ];
```
